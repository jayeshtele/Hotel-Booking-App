# StayNest Hotel Booking App

A responsive Airbnb-inspired hotel booking frontend built with React, Vite, Tailwind CSS, React Router, Redux Toolkit, and RTK Query.

## Features

- Home, explore, stay details, wishlist, trips, host, and not-found routes
- Responsive hotel search, category filters, amenity filters, price range, and sorting
- Wishlist and trip reservations persisted in browser storage
- Live keyless API integrations for weather and currency rates
- Custom favicon, route-aware browser titles, Tailwind design system, and lucide icons

## Run Locally

```bash
npm install
npm run dev
```

## Verify

```bash
npm run lint
npm run build
```
